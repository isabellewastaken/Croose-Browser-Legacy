# Croose-Browser
A browser utilising the WebBrowser component. Intended for Windows Vista to Windows 8. (Windows 8.1 can handle the modern browser.)

This port was authorised by the original creator.
Note that not every website will work with the browser due to it being an old component probably using some form of whatever IE comes with your system. For Vista, that's  IE 7-9. For 7, that's IE 10-11.

There is no XP support.

For the official repository --> https://github.com/her0347/Croose-Browser
For the official modern website (downloads, etc) -->  https://tinyurl.com/croosebrowser
